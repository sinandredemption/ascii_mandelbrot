# ASCII Mandelrot
Navigate through Mandelbrot set right in the terminal

### Installation
`g++ -lncurses -O3 -DNDEBUG main.cpp`

### Usage
**w/a/s/d** to navigate

**Arrow keys** to zoom in/out

**[ ]** to increase/decrease contrast
